2.6K star！速度达30M/S，这款开源下载神器绝了！

由 哟喂 9 月 27, 2024 0 评论

* 戳上方蓝字“开源先锋”关注我



推荐阅读：

《1.1K star!基于Llama3几秒钟完成一本书，这个项目有点厉害！》

大家好，我是开源君

在这个信息爆炸的时代，视频资源遍地开花，但想下载个视频却总是遇到各种限制。不是需要会员，就是格式复杂得让人头疼。作为一名程序员，白天忙着敲代码，晚上还得跟这些视频资源斗智斗勇，简直心累。

今天开源君给大家分享一个超级赞的开源项目 – MediaGo，这货不只是个视频下载器，简直是个视频下载界的“瑞士军刀”。

项目简介

MediaGo是一款开源免费的视频在线提取工具，专门针对m3u8格式的流媒体视频。它的界面简洁，操作直观，即使是技术小白也能轻松上手。

项目是由一群热爱技术的开发者用Electron+Antd+Vite精心打造。它在GitHub上可是个明星项目，标星数高达2.6k，项目下载量更是惊人，达到了199k！

性能特色

  • 无需抓包：内置浏览器，一键嗅探视频资源，再也不用一边看视频一边抓包了。
  • 跨平台支持：无论是Windows、Mac还是Linux，MediaGo都能完美运行，让你的视频下载无界限。
  • 批量下载：高效利用带宽，一键下载多个视频，省时又省力。
  • 移动播放：电脑下载，手机观看，无缝切换，随时随地享受高清视频。
  • 智能识别：自动过滤广告，精准识别视频资源，让你的下载体验更纯净。

快速安装使用

使用MediaGo非常简单，运行下面的代码，就可以启动MediaGo

bash
# 代码下载
git clone https://github.com/caorushizi/mediago.git

# 安装依赖
pnpm i

# 开发环境
pnpm dev

# 打包运行
pnpm release

当然也可以直接使用官方的打包版本，下载适合安装包，一键安装。

项目体验展示

工具支持视频自动嗅探和手工嗅探，操作很简明。

以自动嗅探为例，

1、选择【资源提取】

2、进入视频的网址

3、开始下载，即可下载视频

经过实测，MediaGo的下载速度非常快，看各位的网络情况吧，这边最大速度能达到30M/S！

还支持将视频转换为音频

MediaGo是一款集高效、便捷、智能于一体的视频下载神器。它不仅解决了视频下载的痛点，还让下载过程变得轻松愉快。无论是技术宅还是小白，都能在这个工具中找到乐趣。
